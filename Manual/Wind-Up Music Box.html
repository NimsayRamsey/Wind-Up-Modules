Keep Talking and Nobody Explodes Mod Wind-Up Music Box

On the Subject of Wind-Up Music Box

/!\ IMPORTANT /!\
This is a module in the Wind-Up series. Refer to Appendix WIND-UP KEY for important notes.

The module holds a single key hole in the top left and 4 music buttons on the bottom.

The music box must be wound fully before it can start playing, and the code can only be entered while it's not playing.
