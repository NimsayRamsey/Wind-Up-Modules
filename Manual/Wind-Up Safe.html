/!\ IMPORTANT /!\
This is a module in the Wind-Up series. Refer to Appendix WIND-UP KEY for important notes.

The module contains a large, 12-sectioned combination safe with a keyhole in the center. Using the key, input a 3 part combination to disarm the module.

RULES:

  • The order of submission goes counter, clockwise, counter.
  • The dials are tracked as 0-11, going clockwise.
  • The code is 3 numbers. Turn to the first number, then second, then remove the key while the dial is set to the third.
  • The red LED will turn on while the combination is being tracked. The green LED will turn on during the third rotation.
  • Remove the key while the green LED is on to submit the combination.
  • If the green LED is off, the module will reset without striking.
  • Inputting more than 3 numbers or going clockwise first will stop tracking your combination and turn the LEDs off.

SOLVE METHODS

The code on the tape will be modified multiple times depending on what ports are located on the bomb.

The conversions must go down the list in order and only done if the TYPE of port is on the module.

If there's no ports, the owner is dumb as rocks. Code stays the same.

  • IMPORTANT: any calculations past 0/11 will loop
  • DVI-D - XOR
    • Convert each decimal number to binary, adding 0s leftside if shorter than 4 bits:
    • XOR step 1 & step 2
    • XOR step 2 & step 3
    • XOR step 3 & step 1
    • IMPORTANT: Numbers higher than 11 will loop around to 0
    • Convert back to decimal
    • EXAMPLE: 9-6-5 --> 3-3-1
  • Parallel - Shift
    • Add the number of batteries to each step
    • EXAMPLE: SN3[6] [9-6-5] --> 3-0-11
  • PS/2 - Wheel
    • The correct code is as follows:
    • A - step 1 + step 2 - step 3
    • B - step 2 + step 3 - step 1
    • C - step 3 + step 1 - step 2
    • EXAMPLE: 9-6-5 --> 10-2-8
  • RJ-45 - Grid - MISSING
  • Serial - M-Shift
    • Shift, but the numbers shift backwards, use the number of ports, and are in 4 number chunks
    • If they reach the edge of their chunk, they'll reverse direction
    • EXAMPLE: SN6[9] [2] --> 1-0-11-0-1-2-1-0-11 --> 11
  • Stereo RCA - Doubles - INCOMPLETE
    • Grab the 3rd and 6th digits in the serial number. These will be M1 and M2.
