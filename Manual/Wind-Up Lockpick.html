On the Subject of Wind-Up Lockpick

Rusty Dusty Four-hole Trouble

/!\ IMPORTANT /!\
This is a module in the Wind-Up series. Refer to Appendix WIND-UP KEY for important notes.

The module holds 4 key holes. Three in a row above the fourth.

  • The top 3 holes can be turned between 4 positions. The bottom hole can be turned to submit the current combo.
  • Each hole, as well as the washer around the submit hole, can have 1 of 3 materials, each with a value:
    • Copper - 1
    • Brass - 2
    • Steel - 3

Follow these rules in the order they are listed. Perform the first action that applies. Any values below 1 or above 4 loop back around:

  1. All 3 turn slots are the same color
    • IF Solve Peg is the same material as the Solve Washer
      • Right turn slot should be set to its own material, then subtract one going left
    • Else
      • Invert the values (copper = 4). Left turn slot should then be set to it's own material, then add one going right
  2. There's more copper parts than brass or steel
    • Set each copper slot to the submit washer + the submit peg. All other slots should be their material plus their position
  3. There's only 2 materials used
    • Copper - Number of batteries mod 4 + 1
    • Brass - 3rd + 6th serial number mod 4 +1
    • Steel - Number of ports mod 4 + 1
  4. All 3 slots are different materials
    • Copper slot should add its position
    • Brass slot is slot 1 plus slot 3
    • Steel slot is itself plus submit washer
  5. If none of the above apply, Take the value of each turn slot and add the submit washer
