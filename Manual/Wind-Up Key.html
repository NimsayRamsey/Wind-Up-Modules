Keep Talking and Nobody Explodes Mod Wind-Up Key

On the Subject of Wind The Key

/!\ IMPORTANT /!\
This is a module in the Wind-Up series. Refer to Appendix WIND-UP KEY for important notes.

The module is disguised as a needy and contains 2 LEDs, a keyhole, and a segment display at the top. The module has 2 phases

PHASE 1*
The display will show the number of module solves** needed to activate phase 2. Each solve ticks this number down by 1.
When the count reaches 0, phase 2 will start and an alarm will sound.
Turning the key will not strike during this phase.

PHASE 2
The lower right light will start to flash and the display will change to a time. You must turn the key at the time exactly, or the bomb will immediatly explode. Turning too early will simply strike.

If no other modules remain, the light will stay on. At this point, You can turn the key to disarm the module at any time.

*If this is the only module on the case, the module will skip straight to phase 2
**Ignored modules will not count towards the countdown
